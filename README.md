> A delegate experience focused API for [Ark Core](https://github.com/ArkEcosystem/core).

## Installation

### Clone

```bash
cd ~/ark-core/plugins
git clone https://github.com/arkx-ecosystem/core-delegate-api
lerna bootstrap
```
